Setup parameters.¶

We want to draw 1000 samples (N); assuming a minimum net worth of 10,000 (ALPHA); BETA is 1.1 and should yield something very close to the 80/20 rule; the random SEED is used to ensure that results are reproducible; BINS is the number of equal-sized groupings that we will use to create the histogram of the samples and make some insights about the distribution.¶

In [91]:
BETA, ALPHA, N, BINS = 10000, 1.1, 1000, 100
SEED = 12345
LOCALES_FAKER = ['it_IT']
DUMMY_DATASET_FNAME = 'faker_networth_pareto_distro.csv'

Initialize pareto generator, drawing samples and order them.¶

In [79]:
# setup distribution
dist = pareto(b = ALPHA, scale = BETA)
# drawing samples
samples = dist.rvs(size = N, random_state = SEED)
# ascending order
samples = np.sort(samples)
# samples contains 1000 extractions according to the Pareto distribution

# make an histogram of samples
# use weights to convert absolute frequencies to relative frequencies
# the count of samples inside each bin is normalized to maximum number of samples, to obtain probabilities
counts, bins = np.histogram(a = samples, 
                            bins = BINS, 
                            weights=np.ones(len(samples))/len(samples))
fit = dist.pdf(bins)
In [83]:
fig, ax = plt.subplots(1, 2, figsize=(15, 5))
ax[0].hist(x = bins[:-1], bins = bins, weights=counts)
ax[0].set_title('Histogram of the samples, along with the probabilities')
# to overlap the curve to the hist, calcutate conversion factor f st. 
# max fit * f = max bins_height -> f = max bins_height / max fit
# https://numpy.org/doc/stable/reference/random/generated/numpy.random.pareto.html
ax[0].plot(bins,fit*max(counts)/max(fit),linewidth=2, color='r')
cprobs = dist.cdf(samples)
ax[1].set_title('CCDF')
ax[1].plot(samples, 1-cprobs)
plt.show()
